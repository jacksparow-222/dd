
Oujda Digital Studio - Next.js (App Router) project
Windows-ready, Vercel-optimized.

Quick start (Windows):
1) Install Node 18 LTS: https://nodejs.org
2) unzip this project and open PowerShell in the project folder
3) run: npm install
4) run: npx tailwindcss init -p   (if tailwind not initialized)
5) run: npm run dev
6) open http://localhost:3000

Environment variables:
- NEXT_PUBLIC_WHATSAPP (e.g. +212710925748)
- NEXT_PUBLIC_CONTACT_EMAIL (e.g. smail.al.harrak@gmail.com)
